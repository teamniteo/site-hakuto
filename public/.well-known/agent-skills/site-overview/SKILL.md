# Hakuto Site Builder

Hakuto is a free, open-source site builder for Claude Code. Clone the repo, build with AI, and deploy to Cloudflare.

## Capabilities

- Build landing pages, blogs, and documentation sites using Astro framework
- Tailwind CSS v4 styling with shadcn/ui components
- Deploy to Cloudflare Workers with preview URLs for pull requests
- AI-optimized with llms.txt and structured documentation
